**🧠 My Solution:** Built an end-to-end ML platform using K-means clustering, Random Forest algorithms, and interactive Tableau dashboards.

- 🎯 87.3% churn prediction accuracy (industry benchmark: 75%)
- 💸 R4.8M+ annual revenue potential identified
- 👥 4 strategic customer segments with actionable insights
- 📊 44,000+ customer records analyzed across 6 datasets
